CCNP training

Who is this for?

The Cisco Certified Network Professional is aimed at individuals who are aiming to be Network Professional in:

  • Network Engineers
  • Network Operations Centre
  • Technical Support Personnel
  • Help Desk Technician

This course is also suited for individuals who have an understanding in the execution and verification of routing protocols in the enterprise network.

Reasons to get certified

The CCNP certificate is recognized within the current IT industry and is seen as one of the most valuable certifications globally.

Do you need a CCNP Certificate?

Yes, Cisco Certified Network Professional (CCNP) certification validates your ability to implement and troubleshoot both LAN and WAN networks. As an individual, you will be recognized as a person who can collaborate with specialists on issues related to voice, wireless, advanced security, and video solutions. It's suitable for people who have previous knowledge and experience within this field. Once the certification is obtained the individual will be recognized as:

  • Network engineer
  • Support engineer
  • Systems engineer
  • Network technician

The examination that is needed to be taken to gain the CCNP Certificate

  • Cisco CCNP 350-401 (Core Exam)
  • Any one of the Concentration Exams

Syllabus

The following 3 models are from the syllabus for the CCNP course:

  • CCNP Route: Implementing IP Routing
  • CCNP Switch: Implementing IP Switching
  • CCNP Tshoot: Maintaining and Troubleshooting IP networks

MCSA training

Who is this for?

The Microsoft Certified Solutions Associate (MCSA) is based on individuals who want to validate a comprehensive set of skills necessary to perform the job role of Systems Administrators.

Reasons to get Certified in MCSA

The MCSA training course provides the skills that are needed to, maintain, and manage users, desktop PCs, and servers running on the Windows 7 and Windows 2008 Server platforms and legacy versions.

Do you need an MCSA Certificate?

Yes, the certificate covers topics that are still used within the current work environment and should only be attempted if an individual wants to gain the knowledge.

Learning Path (modules that need to be covered before attempting the certification)

  • Windows Server 2016: Manage, Monitor, and Maintain Servers
  • Windows Server 2016: DNS
  • Windows Server 2016: DHCP
  • Windows Server 2016: IP Addressing
  • Windows Server 2016: Remote Access Solutions
  • Windows Server 2016: Advanced Networking Features
  • Windows Server 2016: Install and Configure Active Directory
  • Windows Server 2016: Active Directory Enterprise Infrastructure
  • Windows Server 2016: Active Directory Certificate Services
  • Windows Server 2016: Implementing Group Policy

The examination that is needed to be taken to gain the MCSA Certificate

  • 70-640 MCTS: Windows Server 2008 Active Directory, Configuring
  • 70-642 MCTS: Windows Server 2008 Network Infrastructure, Configuring
  • 70-646 MCTS: Windows Server 2008, Server Administrator
